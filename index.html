3-Week Game Jam Project

Technical Art Portfolio

During this project, I worked on a wide range of technical and artistic aspects of the game, including VFX, shaders, post-processing, lighting, UI integration, scripting, optimization, and asset integration. My task was to improve gameplay readability while enhancing the visual atmosphere and overall visual consistency of the project.

What I Contributed to the Project

A quick overview of the main responsibilities I handled during production.

⚡
Gameplay VFX

I created several gameplay-focused visual effects to improve player feedback, including an electrical effect for circuit breakers, a smoke effect, and a popcorn effect.
◈
Rendering & Shaders

Developed outline shaders, HDRP Custom Passes, and a vintage television post-processing effect to improve readability and atmosphere.
⌘
Scripting

Developed and modified gameplay scripts such as flashlight battery, light on/off, flickering systems and monster shadow behavior.
☰
UI Integration

Integrated menus, animated transitions, settings panels, quest displays, and automatic keyboard/gamepad UI switching.
🛠
Tools

Created editor tools to optimize workflow, including a procedural generation system, a mesh creator, and a 3D noise texture generator.
◆
Asset Integration

Integrated assets into the project, adapted them to the environment, and ensured they aligned with the artistic direction.

☠
Monster Animation Integration

Integrated the monster's animations into Unity, configured the Animator state machine, and connected animations to gameplay logic.

▱
Optimization

Optimized the scene with LOD setups and culling systems across the level to improve performance.

✦
Lighting & Atmosphere

Handled the complete lighting setup and visual atmosphere to create a dark horror atmosphere while maintaining gameplay readability.

My Technical Artist Approach

Connecting artistic intent, technical constraints, and gameplay feedback.

01 Gameplay

Identify what the player needs to understand immediately.

02 Visual Design

Create a visual effect, lighting setup, or UI element that makes the action clear and readable.

03 Technical Implementation

Integrate everything into Unity using flexible systems that are easy to iterate on and maintain.

Role Summary

Throughout this project, I was responsible for a significant portion of the game's technical presentation, including visual effects, shaders, post-processing, UI systems, lighting, atmosphere, asset integration, animation integration, gameplay support scripting, and optimization.
