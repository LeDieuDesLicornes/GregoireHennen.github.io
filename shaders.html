Post-Process : Filtre Vieux Cinéma (HLSL / HDRP)

J'ai développé un shader de Post-Process personnalisé sous Unity HDRP permettant de recréer l'esthétique d'une pellicule de cinéma vintage dégradée.

1. Conversion Noir & Blanc

Le shader calcule d'abord la luminance de la scène à l'aide d'un produit scalaire (dot product) avec les coefficients standards de perception visuelle (Rec. 601).

Une simple condition basée sur une variable float "_IsBlackAndWhite" permet d'activer ou de désactiver dynamiquement le mode.

// 1. Image de base & Conversion
float3 color = SAMPLE_TEXTURE2D_X(_InputTexture, sampler_InputTexture, uv).rgb;

if(_IsBlackAndWhite > 0.5) {
    float lum = dot(color, float3(0.299, 0.587, 0.114));
    color = float3(lum, lum, lum);
}

2. Effet de Flicker (Saut de lumière)

Pour simuler les variations d'intensité lumineuse d'un vieux projecteur, j'utilise une fonction de hachage (`hash`) basée sur le temps.

L'utilisation de la fonction "floor()" permet de bloquer le bruit sur des paliers nets. La fonction "pow()" accentue la rareté et la brutalité des baisses de lumière selon la variable "_FlickerSharpness" choisie par l'utilisateur.

// 2. EFFET DE FLICKER
float rawNoise = hash(float2(floor(time * _FlickerSpeed), 123.45));

float flickerNoise = pow(rawNoise, _FlickerSharpness); 
float flicker = flickerNoise * _FlickerIntensity;

color *= (1.0 - flicker);

3. Génération de Rayures Procedurales

Les rayures verticales de la pellicule sont créées en découpant l'écran en colonnes via une grille UV unidimensionnelle "gridX".

Si une colonne passe le test de probabilité aléatoire, une ligne est tracée. On génère ensuite des interruptions horizontales via un "breakModifier" pour éviter d'avoir de bêtes lignes continues et droites. Enfin, l'effet applique soit une ombre portée (rayure noire), soit une brûlure (rayure blanche).

// 3. EXTRAIT DES RAYURES
float scratchTime = floor(time * _ScratchSpeed);
float gridX = uv.x * (_ScratchResolution * 5.0); 
float cellX = floor(gridX);
float randCol = hash(float2(cellX, scratchTime));

if (randCol > 0.997) { 
    // Calcul de la longueur et de l'épaisseur
    float xSub = frac(gridX) - 0.5;
    float lineProfile = pow(saturate(1.0 - abs(xSub) * 2.0), thicknessPower);
    scratchLine = lineProfile * withinLine * _ScratchIntensity;
}

4. Grains & Poussières de Lentille

Au lieu d'utiliser une texture de bruit en boucle, le grain est entièrement mathématique et procédural.

Il tourne sur un algorithme de bruit cellulaire à l'aide d'une double boucle "for" qui vérifie les cellules voisines. Cela permet de générer des micro-points et des agrégats de poussière uniques à chaque frame, assurant une dynamique ultra-réaliste.

// 4. BOUCLE DU BRUIT CELLULAIRE
for(int y=-1; y<=1; y++) {
    for(int x=-1; x<=1; x++) {
        float2 neighbor = float2(x, y);
        float2 cell = i_uv + neighbor;
