Script: Flashlight Battery & Mechanics (C#)

This system manages the flashlight's battery life, flickering behavior, and recharging mechanics using a state-machine approach.

1. Dynamic Flickering

The script uses Coroutine sequences to simulate power loss. By analyzing the current battery percentage against a List of thresholds, it triggers randomized blackout durations and intensity drops.


private IEnumerator FlickerRoutine(FlickerSettings settings) {
    _isFlickering = true;
    yield return new WaitForSeconds(Random.Range(settings.minDelay, settings.maxDelay));
    // ... logic for intensity loops
    _isFlickering = false;
}

2. Recharge Logic

The reload system handles the transition from "empty" to "fully charged", triggering specific audio loops and enabling player capabilities (like the flashbang) once the battery hits a specific threshold.


public void ReloadBattery() {
    if (currentBattery < _maxBattery) {
        currentBattery += _batteryGainPerSecond * Time.deltaTime;
        if (currentBattery >= reloadMinimumForFlashBang) player.canFlashBang = true;
    }
}
