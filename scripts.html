Flashlight Battery & Energy Management System

Ce script gère l'intégralité du cycle de vie énergétique de la lampe torche. Au-delà d'une simple gestion de valeur (0-100%), le système utilise des Coroutines et des structures de données personnalisées pour coupler mécaniques de gameplay et retour visuel intuitif.

1. Système de Clignotement Dynamique

Pour éviter une interface trop chargée, l'état de la batterie est communiqué via un comportement instable de la lumière. Le script utilise un tableau de FlickerSettings configurable via l'inspecteur, permettant de définir des comportements distincts par paliers de batterie.


private void CheckForRandomFlicker() {
    foreach (var config in _flickerConfigs) {
        if (currentBattery > config.minBatteryThreshold && currentBattery <= config.maxBatteryThreshold) {
            StartCoroutine(FlickerRoutine(config));
            return;
        }
    }
}

private IEnumerator FlickerRoutine(FlickerSettings settings) {
    _isFlickering = true;
    yield return new WaitForSeconds(Random.Range(settings.minDelay, settings.maxDelay));
    
    // Séquence de clignotement aléatoire
    int flickers = Random.Range(3, 6);
    for (int i = 0; i < flickers; i++) {
        SetLightIntensity(Random.Range(settings.flickerMinIntensityPercent, 0.4f));
        yield return new WaitForSeconds(Random.Range(0.05f, 0.15f));
        SetLightIntensity(1.0f);
        yield return new WaitForSeconds(Random.Range(0.05f, 0.1f));
    }
    _isFlickering = false;
}

2. Mécanique de Recharge (ReloadBattery)

La méthode ReloadBattery() utilise une logique d'états pour gérer la progression. Elle vérifie si le joueur vient d'initier la charge (déclenchement des sons) et ajuste dynamiquement l'intensité lumineuse et les propriétés du joueur (comme le FlashBang) une fois certains seuils atteints.


public void ReloadBattery() {
    if (!_isReloading && currentBattery < _maxBattery) {
        _isReloading = true;
        startCharginMetalLoopAudioSource.Play();
        if (_noiseEmitter != null) _noiseEmitter.PlayNoiseWithDelay(2.3f);
    }

    if (currentBattery < _maxBattery) {
        currentBattery += _batteryGainPerSecond * Time.deltaTime;
        if (currentBattery >= reloadMinimumForFlashBang) player.canFlashBang = true;
    }

    if (currentBattery >= _maxBattery) {
        currentBattery = _maxBattery;
        if (!_wasFullyCharged) {
            _wasFullyCharged = true;
            fullyChargedAudioSource.Play();
        }
    }
}
